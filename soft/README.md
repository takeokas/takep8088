# program loader via Xmodem  
## プログラムについて  
- nasm でアセンブル可能  
- xmodemプログラムは、fff00h に置く。ROMにはそうなるように書きこむ  
- ロードされるバイナリは 00400h から置かれる  
- ロードが終了すると、00400h 番地へ jmp {cs=40h,pc=00h}
- xmodem プログラムのスタックは、020000h から、前へ伸びる  
- プログラムが、スタートすると「*」を1つだけUSARTから送信

## 使い方
- ホスト・マシンで xmodem 送信を起動後、takep8088 をリセットすると、xmodem 通信が始まる
  - 「*」がホストに送られるが、問題ない  
- xmodemが開始していない間、SW1 を押下すると、LED3 が点灯する
