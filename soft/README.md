# program loader via Xmodem  
## プログラムについて  
- nasm でアセンブル可能  
- xmodemプログラムは、fff00h に置く。ROMにはそうなるように書きこむ  
- xmodem は  
  - プログラムがスタートすると「*」を1つだけUSARTから送信  
  - ロードされるバイナリは 00400h{0040h:0000h} から置かれる  
  - ロードが終了すると、0040h:0000h 番地へ far jump {cs=0040h,pc=0000h}
- xmodem100 は  
  - プログラムがスタートすると「#」を1つだけUSARTから送信  
  - ロードされるバイナリは 00500h から置かれる  
  - ロードが終了すると、0040h:0100h 番地へ far jmp {cs=0040h,pc=0100h}
- xmodem プログラムのスタックは、020000h から、前へ伸びる  

## 使い方
- ホスト・マシンで xmodem 送信を起動後、takep8088 をリセットすると、xmodem 通信が始まる
  - 通信速度は 9600 bps。 J1 ジャンパは、真ん中(3-4)を選択  
  - 「*」または「#」がホストに送られるが、問題ない  
- xmodemが開始していない間、SW1 を押下すると、LED2 が点灯する
